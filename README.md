# picoCTF 2019

picoCTF2019 was a two-week long CyberSecurity challenge created by security experts at Carnegie Mellon University. The 2019 competition was held between September 27, 2019 and October 11, 2019. My Team __AeroCoders__ secured __Global Rank 189__ in the competition.
